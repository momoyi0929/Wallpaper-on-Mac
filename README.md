# Wallpaper-on-Mac
1.从Wallpaper Engine 选择video/sence种类壁纸导出
2.video类型不做更改
3. sence需要利用RePKG解包从而得到video。方法： path/RePKG.exe extract path/filename.pkg -o directoryname_path
4.打开Mac的Dynamic Wallpaper
5. 导入本地视频文件
